Welcome to Firmware Forge ESP8266 Updater's documentation!
==========================================================

Welcome and thank you for choosing Firmware Forge! To get started, please visit the getting started page.
For information on how the updater communicates with the server, see the how it works page. For general information
regarding the updater API, please see ESP8266 Updater. Advanaced usage will cover how to store data from the updater
in flash memory so that settings persist across ESP deep sleep and power on resets, these features will come in a later update.
